# Sound Effects

Place the following audio files in this directory:

- `roll.mp3` - Sound of dice rolling
- `win.mp3` - Sound for a normal win
- `jackpot.mp3` - Sound for a Leopard/Jackpot win
- `ambiance.mp3` - Background casino ambiance (looping)
- `click.mp3` - UI click sound (optional)

Note: The application uses the native HTML5 Audio API. Supported formats include MP3, WAV, and OGG.
